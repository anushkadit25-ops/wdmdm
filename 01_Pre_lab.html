Pre-Lab: Installation and Configuration of VS Code & Live Server

Objective

To study the installation and configuration of Visual Studio Code (VS Code), install the Live Server extension, and create a standard project directory structure for frontend web development.

Step 1: Download VS Code

Open any web browser and visit the official Visual Studio Code website. Download the installer according to your operating system.

Step 2: Install VS Code

Run the downloaded setup file. Click on Next, accept the license agreement, choose the installation location, and click on Install. After installation, click Finish.

Step 3: Launch VS Code

Open Visual Studio Code from the Start Menu or Desktop shortcut. The welcome screen will appear.

Step 4: Install Live Server Extension

Click the Extensions icon on the left sidebar. Search for Live Server and click the Install button.

Step 5: Create Project Folder

Create a folder named Frontend Project. Inside this folder, create the following files:

  • index.html
  • style.css
  • script.js
  • images (folder)

Step 6: Run the Project

Open the index.html file, right-click anywhere inside the editor, and select Open with Live Server. The webpage will automatically open in your default web browser.

Conclusion

Visual Studio Code was successfully installed and configured. The Live Server extension was added successfully, and a standard frontend project structure was created. The project can now be developed and tested efficiently using Live Server.
